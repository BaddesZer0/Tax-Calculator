Program Hakkında:

Bu program herhangi bir satın alım yaptığınızda ürünün ham değerine ek olarak %18 KDV eklenmiş toplamda ödenmesi gereken ücret tutarını hesaplamaktadır.

Nasıl Çalıştırılır?:

1)Satın alım yaptığınız ürünün vergisiz fiyatını giriniz.

2)%18 KDV eklenmesi ile oluşan ödemeniz gereken toplam miktar ekranda görüntülenecektir.
